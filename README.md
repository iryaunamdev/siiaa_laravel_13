# SIIAA - Sistema Integral de Información Académica y Administrativa

Proyecto desarrollado en Laravel 13 como núcleo del sistema institucional del IRyA.

## Tecnologías
- Laravel 13
- Livewire
- MariaDB
- Redis
- Docker (Sail)

## Estado
En desarrollo activo.

## Objetivo
Implementar una plataforma modular, escalable y centralizada para la gestión académica, administrativa y de investigación.

## Notas
Este repositorio representa la implementación base del sistema, independiente de versiones anteriores.
